property information.

  • Flex Container: The parent element that contains flex items. It's defined using the display: flex; or display: inline-flex; property.
  • Flex Items: The child elements of a flex container. They are laid out along the main axis of the flex container.
  • Flex Wrap: Specifies whether flex items should wrap onto multiple lines or stay on a single line. It can be set using the flex-wrap property with values like nowrap, wrap, or wrap-reverse.
  • Flex Direction: Determines the main axis direction. It can be set using the flex-direction property with values like row, row-reverse, column, or column-reverse.
  • Justify Content: Aligns flex items along the main axis. It can be used to distribute space between items or to align them to the start, end, or center. Properties include justify-content: flex-start, justify-content: flex-end, justify-content: center, justify-content: space-between, and more.
  • Align Items: Aligns flex items along the cross axis. It controls how items are positioned vertically (if the main axis is horizontal) or horizontally (if the main axis is vertical). Properties include align-items: flex-start, align-items: flex-end, align-items: center, align-items: stretch, and more.
